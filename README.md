# NewValue Server

Backend project using **Node.js**, **Express**, and **SQL (MySQL) with Sequelize**.

## Features

- Express server
- SQL database connection with Sequelize
- CRUD routes for Users (`/api/users`)
- Environment variables configuration
- CORS enabled
